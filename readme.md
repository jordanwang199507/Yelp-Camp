# YelpCamp

## Live Demo
to view the live demo of this app, please visit https://warm-shore-55365.herokuapp.com/campgrounds

## Features
- Authentication
- Authorization
- Manage campground posts with basic functionalities: Create, Edit, Delete posts
- Flash message response
- Fuzzy Search
- User Profiles
- Image Upload

## Built With
### Front-end
- ejs
- bootstrap
### Back-end
- express
- monogodb
- mongoose 
- passport
- passport local
### Platforms
- Heroku
- Cloudinary
